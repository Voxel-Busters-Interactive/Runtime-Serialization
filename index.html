Runtime Serialization provides powerful, efficient way to serialize and deserialize objects.


Highlights:

• It’s fast and efficient.

• Uses minimal memory allocations.

• Extend support to external classes by implementing Extensions.

• Serialize any custom object including Unity components like GameObjects, Transforms, Monobehaviour, Colliders etc.

• Supports Scene serialization.

• Allows saving data to PlayerPrefs / Files.

• Provides serialization callbacks.
Includes source code with Documentation
Supports PC, Mac, Web, iOS, Android, Windows Phone and Windows Store platforms. Works with Unity 4 and above.

Note:
• Metro/WP8 platforms doesn’t support Hashtable and Arraylist classes so we are not supporting serialization of mentioned types.
