 ▼NRuntimeSerializationThe VoxelBusters.RuntimeSerialization namespace contains classes that can be used for serializing and deserializing objects at runtime. Serialization is a process of converting an object into binary stream of data and storing it on disk. Deserialization is a process of reading stored data and recreating objects from it
